# Django Application with Celery Beat

## Overview

This Django application uses **Celery** for asynchronous task processing and **Celery Beat** to schedule periodic tasks.

---

## Requirements

- Python 3.x
- Django
- Celery
- Redis (or other broker supported by Celery)
- django-celery-beat
